#include <iostream>

Topic: Arithmetic Operations

Question: Write a program to add, subtract, multiply, and divide two numbers.

Code:

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5;
    cout << "Sum: " << (a + b) << endl;
    cout << "Difference: " << (a - b) << endl;
    cout << "Product: " << (a * b) << endl;
    cout << "Quotient: " << (a / b) << endl;
    return 0;
}

Output:

Sum: 15
Difference: 5
Product: 50
Quotient: 2
